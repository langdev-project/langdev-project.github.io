Home

Welcome to the official homepage for the LANGDEV Project!! The LANGDEV Project is a project centered around the creätion and development of fictional languages and the tools needed to record them properly. The LANGDEV Project was started by Gô SHOEMAKE and is still largely a personal project on their account.

Language is meant to be shared, and that is why all of the language data associated with the LANGDEV Project is released under a CC0 1.0 Universal Public Domain Dedication, and many of the tools and other information associated with the project are available under open licenses as well.
