Tools

IETF Language Tags

The LANGDEV Project provides canonical private-use language subtags for marking its languages and scripts.

LexisML Index Records

LREC is a file format designed by LANGDEV for encoding a dictionary index in an easy, readable, plain-text form.

LANGDEV in Unicode

LANGDEV in Unicode specifies a number of character properties for a subset of the Unicode Private-Use Areä, for use with LANGDEV scripts.

Unifont LANGDEV

Unifont LANGDEV is a font project built upon GNU Unifont to provide a simple fallback font for LANGDEV scripts.
